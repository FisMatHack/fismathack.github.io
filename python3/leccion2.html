Operadores aritmeticos en Python


+ -> suma | - -> resta | % -> resto(el resto de la división) | / -> división | // -> dividir de forma entera | * -> multiplicación | ** -> exponenciación(un número elevado a la potencia de otro)
1 + 1 = 2, en Python exite la función "print" el cuál imprime texto en la consola.
print(1+1)
La salida del código sería 2, ya que 1+1=2, print se encarga de imprimir.
Cuándo un número es dividido como entero esté es redondeado.
3 / 8 = 0.375
3 // 8 = 0
El número es redondeado a 0,ya que esté no pasa la siguiente regla: número > 0.500.
