Headless - Machine HackTheBox

In Headless, there is a website running that has a contact section. By trying to perform an XSS attack on the "Message" field, I can trigger an XSS alert and modify some of the headers to get an XSS and steal the victim user's session cookie. Once we have a cookie, we can use it in /dashboard, which has a command injection vulnerability in the "date" parameter of the request sent by POST. The escalation consists of creating a Bash file and giving it execution permissions.
